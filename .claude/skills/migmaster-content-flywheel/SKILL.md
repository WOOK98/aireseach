# MIG MASTER Content Flywheel

One seed in → four channel-ready assets out. Store: migmaster.shop (Shopify, gamepro-master theme, dark gaming aesthetic). Niche: Nintendo Switch accessories, flagship interest cluster around "mig switch", "mig dumper", backup/flashcart-adjacent hardware.

## Known baseline (update as data changes)
- GSC: ~249 impressions, 0 clicks → titles/meta are the bottleneck, not just rankings. Every asset's title must be written as a CTR play.
- Identified keyword gaps: "mig switch v2", "mig dumper buy". Check current GSC data for new gaps before each cycle if Wook provides it.

## Compliance guardrail (check EVERY cycle)
This niche borders on console-modding/backup-device territory. Content must market the store's actual legal products and frame use cases as lawful (playing/backing up games you own, homebrew, accessories). Never write content that instructs or encourages piracy, downloading ROMs of games the reader doesn't own, or circumventing purchases — both for legal exposure and because Reddit/Google will bury it. If a seed topic can't be covered without that framing, tell Wook and propose an adjacent angle.

## The flywheel — per seed topic

**Input**: one seed (a product, a keyword gap, a Switch-scene news event, or a customer question).

**Step 1 — Angle brief (do this first, 10 lines max)**: target keyword + search intent, the one question the piece answers, target reader (Switch owner researching hardware), and the CTR-optimized title + meta description (title ≤60 chars, meta ≤155, must contain a concrete hook: price, compatibility, "v2 vs v1", etc.). Confirm angle with Wook only if the seed is ambiguous; otherwise proceed.

**Step 2 — Pillar asset (store blog/landing page)**: 800–1500 words, HTML ready for Shopify blog. Structure: answer the query in the first 100 words → comparison table or spec table → FAQ section (4–6 questions, written to match real query phrasings) → internal links to 2–3 product pages → single CTA. Include JSON-LD (Article + FAQPage; Product schema if a product page). Match the dark-gaming brand voice: direct, enthusiast-to-enthusiast, no corporate filler.

**Step 3 — Reddit post (English)**: pick the subreddit by topic fit and check its self-promo rules in the moment. Format: genuinely useful standalone content (guide/comparison/PSA), store mention only where natural or in comments per sub norms. Never copy the blog post — rewrite natively for Reddit voice. Flag to Wook that account karma/history matters more than post quality for survival.

**Step 4 — X 中文帖**: 1–3 tweets. 面向中文圈的 Switch 玩家/数码爱好者,信息点前置(兼容性、对比、价格),配图建议(产品图或对比表截图),链接放最后。参考 Wook 既有的中文 X 内容风格:短句、具体、不堆 emoji。

**Step 5 — Blogger 长文 (optional, when seed has search volume in zh)**: 中文版 pillar 文章,不是翻译而是为中文搜索意图重写。

**Output format**: deliver all assets in one response, each in its own clearly labeled section, HTML in code blocks ready to paste. End with a publish checklist (where to paste each, what to verify in GSC after 2 weeks: impressions on target keyword, CTR change).

## Cadence & measurement
- Suggested rhythm: 1 seed/week. Keep a running list of published seeds + target keywords so cycles don't cannibalize each other.
- Each cycle, if Wook provides fresh GSC/Shop Campaigns data, open with a 5-line read: what moved since last cycle, and let that pick the next seed.
